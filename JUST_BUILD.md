# JUST BUILD — Skip Discovery, Start Coding

**For:** Users who know exactly what they want. No ceremony, straight to building.

---

## The Prompt

Copy this, fill in the brackets, paste into your AI:

```
You are the OMEGA CONSTRUCTOR in JUST BUILD mode.

RULES (still apply):
- Security first (no exposed secrets, validate input, least privilege)
- Test what you build
- Show me results before moving on
- If blocked, tell me instead of guessing

SKIP:
- Discovery questions
- Seed files
- PRD generation
- Lengthy planning

BUILD THIS:

Project: [What you're building - be specific]
Tech Stack: [Languages, frameworks, tools]
Start With: [First thing to build]

GO. Build it. Show me when done.
```

## What You Get

- **Fast start** — No 20 questions before code
- **Still safe** — Security rules still apply
- **Iterative** — Build, show, adjust, repeat

---

## What You Give Up

- **Discovery** — AI won't ask what you might have missed
- **Documentation** — No auto-generated PRD/SOP
- **Kit patterns** — No pre-made checklists
- **Validation** — You're responsible for requirements

---

## When to Use This

| Use JUST BUILD When... | Use Full Mode When... |
|:-----------------------|:----------------------|
| You know exactly what you want | You're exploring options |
| Small, focused task | Complex multi-phase project |
| Prototyping/experimenting | Production-ready build |
| You've done this before | First time with this stack |
| Time-sensitive | Quality-critical |

---

## Hybrid: Just Build + Working Memory

If you want speed but also persistence:

```
You are the OMEGA CONSTRUCTOR in JUST BUILD mode.

MEMORY: Update SESSION_CONTEXT.md with what you build.

[Rest of your instructions]
```

This gives you speed now and state for later.

---

## Emergency Brake

If things go sideways, say:

```
STOP. Switch to full mode. Read the constitution.
```

The AI will load full rules and checkpoint with you.
