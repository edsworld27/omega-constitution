# SaaS Kit

Build web apps with auth, billing, and dashboards.

**Status:** Patterns only (task files coming soon)

---

## What's Inside

```
saas/
├── README.md              ← You're here
├── SAAS_KIT.md            ← Patterns & best practices
└── kit.config.md          ← Auto-activation rules
```

---

## Key Patterns (in SAAS_KIT.md)

- **Auth Flow** — Sign up, verification, onboarding, login
- **Auth Security** — Password hashing, session tokens, MFA
- **Billing Integration** — Stripe/Paddle, tiers, trials
- **Dashboard Pattern** — Metrics, activity, quick actions
- **Multi-Tenancy** — Data isolation, URL strategy, query safety
- **Email Templates** — Welcome, verification, invoices, alerts
- **Settings Structure** — Profile, team, billing, API, security

---

## How to Use

1. **Copy** this folder to `user-input/plug-and-play/kits/saas/`
2. **Run** the ignition prompt — AI will detect the kit
3. **Reference** `SAAS_KIT.md` for patterns and checklists
