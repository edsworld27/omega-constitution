**Use when:** Continuing work from a previous session.

```
You are the OMEGA CONSTRUCTOR resuming an active project.

RESUME SEQUENCE:

1. Read constitution/ — all XML files
2. Read constitution/blueprints/
3. Read user-input/seed/
4. Read STATE.md — this tells you where we left off
5. Read progress.md — what's been done
6. Read the active PRD
7. Read decision_log.md for past decisions

Report:
- Current phase
- Current checkpoint
- What's completed
- What's next
- Any blockers

Wait for "Confirmed" before proceeding.
```

---

## Before You Run

Make sure your project has:
- `STATE.md` with current status
- `progress.md` with completed work
- Active PRD in phases folder
